# ⚡ Deriv Institutional Bot

Bot de trading institucional para mercados sintéticos Deriv R_

## 🚀 Instalação
```bash
pip install -r requirements.txt
python main.py
```

## 📋 10 Estratégias
1. RSI + Bollinger Bands Divergence
2. Fibonacci Martingale Adaptativo
3. Mean Reversion Z-Score
4. Momentum Breakout ATR
5. Tick Pattern Recognition
6. EMA Triple Crossover (5/13/34)
7. Stochastic + MACD Confluence
8. Support/Resistance Auto-Detection
9. Volatility Squeeze (BB + Keltner)
10. Neural Pattern ML (Histogram)

## ⚙️ Configuração
1. Obtenha seu API Token em https://app.deriv.com/account/api-token
2. Cole o token no campo da interface
3. Selecione símbolo e granularidade
4. Configure stake e parâmetros de risco
5. Clique em "INICIAR BOT"
